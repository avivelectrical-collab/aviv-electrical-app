# 📋 תוכנית מערכת תמחור חכמה - ERP מתקדם לקבלנות חשמל

## 🎯 מטרת הפרויקט
מערכת תמחור אוטומטית המעדכנת מחירון ספקים מחשבוניות, מזהה שינויים, מתריעה על חריגות, ומשווה למחירון דקל.

---

## 📊 סקירת נתונים קיימים

### ✅ מה שיש כבר (מסשנים A-C):
- **supplier_invoices** - חשבוניות עם AI parsing
- **cashflow_suppliers** - נתוני ספקים מסודרים
- **Excel system** - מערכת תמחור עם תעריפי דקל
- **Senzey integration** - הכנסת הצעות מחיר

### 🎯 מה שנוסיף:

- **comparison_engine** - השוואה דקל vs. מציאות

## 🚀 שלבי פיתוח - 5 סשנים

### **📦 סשן D-1: מחלץ פריטים מחשבוניות (AI Enhanced)**
**מטרה:** הרחבת ה-AI לזיהוי פריטים מפורטים בחשבוניות

**פיצ'רים:**
- הרחבת prompt AI לזיהוי: `itemName, itemCode, quantity, unitPrice, totalPrice`
- יצירת `supplier_items` collection בFirebase
- ממשק אישור פריטים (כמו SupplierScanApproval)
- זיהוי קודי פריטים סטנדרטיים (MAT-XX, קטלוגים)

**מבנה נתונים:**
```js
supplier_items: {
  id, supplierId, supplierName, invoiceId, 
  itemName, itemCode, category, 
  quantity, unitPrice, totalPrice, 
  date, confidence, approved, notes
}
```

**תוצר:** כל חשבונית מפורקת לפריטים מאושרים עם מחירים

---

### **📈 סשן D-2: מנוע מעקב מחירים (Price Tracking)**
**מטרה:** מעקב היסטורי מחירים וזיהוי שינויים

**פיצ'רים:**
- `price_history` collection - היסטוריה של כל פריט
- זיהוי אוטומטי פריטים זהים (לוגיקת matching)
- חישוב שינויי מחיר: אחוזים, מגמות
- דשבורד "מעקב מחירים" עם גרפים

**מבנה נתונים:**
```js
price_history: {
  itemKey, // hash של itemName+itemCode
  prices: [{date, price, supplierId, invoiceId, quantity}],
  avgPrice, minPrice, maxPrice, trend, lastUpdate
}
```

**תוצר:** מסד נתונים עם מחירים היסטוריים וזיהוי מגמות

---

### **🚨 סשן D-3: מנוע התראות (Alert System)**
**מטרה:** התראות אוטומטיות על שינויים חשובים

**פיצ'רים:**
- זיהוי שינויי מחיר >15% (סף מותאם)
- התראות פריטים חדשים שלא היו
- התראות ספקים חדשים
- דשבורד התראות עם פילטרים
- התראות email/SMS (אופציונלי)

**סוגי התראות:**
- 🔴 מחיר קפץ >20%
- 🟠 מחיר ירד >15% (הזדמנות)
- 🆕 פריט חדש התווסף
- ⚠️ פריט לא נראה 3+ חודשים

**תוצר:** מערכת התראות פרו-אקטיבית

---

### **⚖️ סשן D-4: מנוע השוואות (Dekel vs Reality)**
**מטרה:** השוואה בין מחירון דקל למחירי ספקים אמיתיים

**פיצ'רים:**
- הרחבת מחירון דקל הקיים
- מיפוי אוטומטי פריטים ← דקל (לוגיקת חיפוש)
- טבלת השוואות: "דקל ₪150 | ספק ₪130 | חיסכון 13%"
- המלצות אוטומטיות: "השתמש במחיר ספק"
- ייצוא דוחות התחרותיות

**ממשק השוואה:**
```
סעיף: כבל NYM 3x2.5
├─ דקל: ₪12.50/מ'      (מרץ 2026)
├─ ראש חשמל: ₪10.80/מ'  (ינואר 2026, avg 3 הזמנות)  ✅ זול 13.6%
├─ דיפו: ₪11.20/מ'      (דצמבר 2025)               ✅ זול 10.4%
└─ המלצה: השתמש בראש חשמל
```

**תוצר:** ממשק השוואות מתקדם עם המלצות

---

### **🎮 סשן D-5: ממשק תמחור חכם (Smart Pricing UI)**
**מטרה:** ממשק משתמש לבניית הצעות מחיר מתקדמות

**פיצ'רים:**
- **בונה פרויקטים:** רשימת סעיפים עם בחירת מחיר
- **בחירה חכמה:** דקל/ספק/מותאם לכל סעיף
- **תצוגת רווחיות:** עלויות vs. הכנסות
- **אופטימיזציה:** המלצות למקסום רווח
- **ייצוא מושלם:** Excel + Senzey + PDF

**ממשק בחירה לסעיף:**
```
[🔧] התקנת לוח חשמל 3P 100A
├─ עבודה: ₪280 (2.8 שעות × ₪100)
├─ חומרים: [בחר מחיר] 
│   ● דקל: ₪850        רווח: ₪420 (33%)
│   ● ראש חשמל: ₪720   רווח: ₪550 (43%) ← מומלץ
│   ● מותאם: [____]    
└─ מחיר ללקוח: ₪1,000
```

**תוצר:** מערכת הצעות מחיר חכמה ואוטומטית

## 💰 ערך עסקי צפוי

### **חיסכון בעלויות:**
- זיהוי ספקים זולים יותר: **5-15% חיסכון בחומרים**
- מיטוב מחירי הצעות: **שיפור שולי רווח**
- חיסכון בזמן תמחור: **80% פחות זמן הכנה**

### **שיפור תחרותיות:**
- מחירים מדויקים יותר = זכיות במכרזים
- מעקב מגמות שוק = החלטות חכמות
- אופטימיזציה אוטומטית = רווחיות מקסימלית
